# Movies API

The Movies API provides endpoints for managing movie resources.

## Overview

This API allows you to create, read, update, and delete movie records.

## Endpoints

### GET /api/v1/movies

Retrieves all movies.

### GET /api/v1/movies/:id

Retrieves a specific movie by ID.

### POST /api/v1/movies

Creates a new movie.

### PUT /api/v1/movies/:id

Updates an existing movie.

### DELETE /api/v1/movies/:id

Deletes a movie.
